Summary: Latitude vs X

The true power of data lies in its ability to answer questions definitively. The purpose of this project was to answer a fundamental question: “What’s the weather like as we approach the equator?”

A sampling of 500+ cities all over the world were randomly generated through pandas. From there, real-time weather data was gathered using a series of successive API calls via Google’s OpenWeatherMap API. The data was then cleaned and assembled to display relevant information (City, Latitude, Longitude, Max Temperature, Humidity, Cloudiness, Windspeed, and Country).

Finally, the clean dataset was plotted via Matplotlib to view how different variables changed as the distance from the equator changed. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations observed.
